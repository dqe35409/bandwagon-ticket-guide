## 什么情况下需要提交工单

先把这个搞清楚，避免白白浪费时间——因为很多问题是不归搬瓦工管的。

搬瓦工卖的是"无管理型 VPS"，意思是他们只负责机器本身正常运行，里面装什么软件、怎么配置、为什么跑不起来，这些都是你自己的事。

**可以提交工单的情况：**

- IP 被封需要付费换 IP（或者想确认 IP 封禁状态）
- 账户问题：登录异常、账户被暂停、忘记密码但邮件打不开
- 机器硬件层面的故障（不是你的程序问题，是机房出了问题）
- 退款申请（需要在购买 30 天内）
- 账单或支付相关的疑问

**不该发工单的情况：**

- SSH 连不上（先检查 IP 是否被封、SSH 端口是否正确）
- 自己安装的软件跑不起来
- 服务器速度"感觉变慢了"（先用测速工具确认，再考虑是否线路问题）

## 搬瓦工提交 Ticket 完整步骤

以下是最新的工单提交流程，按照当前后台界面整理：

**第 1 步：登录账户，进入工单系统**

登录搬瓦工官网后台（Client Area），点击顶部菜单的 **Support**，下拉菜单里找 **Tickets**，点进去。

也可以直接访问工单提交页面：

👉 [前往搬瓦工工单系统](https://bwh81.net/aff.php?aff=74585)

**第 2 步：创建新工单**

进入工单列表页后，点击 **Open New Ticket** 按钮，进入工单填写界面。

**第 3 步：填写工单内容**

工单页面包含以下几个字段：

- **Related Service（相关服务）**：选择这次问题涉及到的具体 VPS，没有对应的就选 None
- **Subject（工单主题）**：用一句话概括你的问题，比如 "My IP seems to be blocked"
- **Message（正文）**：详细描述问题，包括时间、现象、你已经尝试过的排查步骤
- **Attachments（附件）**：有截图、日志文件可以上传，没有就不管

**第 4 步：点击 Submit 提交**

确认内容无误后，点 Submit。提交后会收到一封确认邮件，之后客服回复时也会邮件通知你。

## 工单必须用英文写——附实用模板

这是很多人卡住的地方。搬瓦工没有中文客服，工单内容**必须用英文**写。不会英文没关系，把你要问的东西用中文写清楚，复制到谷歌翻译或者 DeepL 翻译一下就行，客服基本都看得懂。

下面给出几个常见场景的工单模板，可以直接改动使用：

**场景一：IP 被封，申请付费更换**


Subject: IP address blocked, request IP change

Dear Support,

My VPS (service ID: XXXXXX) seems to have a blocked IP address. 
I have verified this using the ping/traceroute tools and cannot connect via SSH.

Could you please help me change to a new IP address? 
I understand there is a fee for this.

Thank you.


**场景二：机器无法开机**


Subject: VPS won't boot after restart

Dear Support,

My VPS (service ID: XXXXXX) failed to boot after I restarted it via KiwiVM panel. 
The status shows "running" but SSH is unreachable and console shows no output.

Could you please check if there is a hardware issue on your end?

Thank you.


**场景三：申请退款**


Subject: Refund request

Dear Support,

I would like to request a refund for my recent order (Order ID: XXXXXX, purchased on MM/DD/YYYY).

The reason is: [briefly explain your reason].

Thank you for your assistance.

## 工单多久有回复？机器人"Martin C"是怎么回事

提交完工单之后，有时候会在半小时内收到一封回复——但回复人叫 **Martin C**，而且内容答非所问。

这就是搬瓦工的 AI 机器人。它的回复里通常会有这样一段话：

> "Note: I am a robot and my reply is based on the information provided in the ticket..."

看到这行字，说明是机器人回了你。机器人解决不了问题怎么办？很简单，把你原来的问题原文复制一遍，回复到这个工单里。这样工单会进入人工处理队列，等真人客服来跟你说话。

**关于回复时间：** 正常情况下，新工单会在几小时到24小时内得到真人回复。但这个时间取决于三个因素——你提交工单的时间（他们按加拿大太平洋时间上班，大概是北京时间凌晨一点到下午两点之间是工作时间）、问题复杂程度、以及当时工单队列的长度。赶上大事件（比如机房故障）那几天，可能要等更久。

提交完耐心等就行。**不要重复提交新工单或者不停回复催促**，那样只会让你的工单排到队伍更后面。

不过从实际使用来看，工单系统的回复效率比邮件稳定得多，建议优先用工单。注意，搬瓦工没有客服电话，也没有微信、QQ 等国内即时通讯支持。

## FAQ：关于搬瓦工提交工单的常见问题

**Q：工单可以用中文写吗？**

不行。搬瓦工只有英文客服，工单必须用英文提交。用翻译软件翻译一下就行，对方能看懂大意就可以。

**Q：提交工单后收到了"Martin C"的回复，是正常的吗？**

是的，Martin C 是搬瓦工的 AI 自动回复机器人。如果回复内容没有解决你的问题，把原始问题再回复一遍，就会转到人工客服处理。

**Q：工单提交后多久能得到真人回复？**

一般是几小时到24小时之间。具体时间取决于搬瓦工的工作时间（按加拿大太平洋时间）和当前工单队列量。复杂问题可能需要更长时间。

**Q：没有账号可以联系搬瓦工客服吗？**

可以。没有账号或者账号登不进去的情况下，可以直接发邮件到 support@bandwagonhost.com。

**Q：工单提交多次没有回复，能催吗？**

可以在原工单里追加一次回复，但不要频繁催促或者重新开新工单。重复操作会让你的工单排名靠后，反而更慢。
